Ejemplo String2.html

El siguiente script rompe una string en dos mitades y las imprime en pantalla. Las mitades serán iguales, siempre que el string tenga un número de caracteres par. En caso contrario, no se podrá hacer la mitad exacta, pero se podrá hacer una aproximación. Para este ejemplo, se usa el método slice().

indexOf(patron, indice)

El método indexOf() retorna el primer índice en el que se puede encontrar un elemento dado en el string, o retorna -1 si el elemento no está presente.

También existe su contraparte lastIndexOf() que busca en el String el último índice en el que se puede encontrar un elemento dado.
